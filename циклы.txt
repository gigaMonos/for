циклы созданы для выполнения одного и того же отрезка кода
Один круг цикла называет литерацией и выполняется если, значение вернет true
инкремент нужен для того, чтобы задать новый круг пока значение возвращает  true
конструкция DO...WHILE делаетс в том случае когда нужно сделать цикл внезависимость от того вернулала переменная  true 
FOR работа цикла
for ( let num = 0; num > 5; num++){
    console.log(num);
}
1. выполняется начало цикла let num = 0;
2. выполняется условие num > 5
3. если условие ПРАВДА то выполняется тело цикла 
console.log(num);
4. выполняется шаг num++ 
повтор начинается со второго пункта
Break деректива которая  останавливает работу досрочно
Continue останавливает часть цикла 
при более сложном варианте использования цикла , например когда внутри цикла находитсядругой цикл, может понадобиться обращение к материнскому циклу с дочернего цикла
 это можно сделать с помощью Меток
